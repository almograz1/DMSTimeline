# DMS Timeline Gantt — 90-Second Demo Script

**Target length:** ≤ 1:30 · **Format:** screen recording + voiceover (or captions)
**Setup:** record on the deployed app (`https://dms-timeline.vercel.app`), 1920×1080, browser at 100%, signed in. Pre-pick names: swim lane `Taichi`, subgroup `HB2 — HeartBeat2`, task `API hardening`, milestone `HB2 GA`. Do one silent dry-run first.

> Voiceover budget ≈ 210 words. Keep it brisk; let the clicks carry the story.

---

### Scene 1 — Create & share (0:00–0:18)
**On screen:** Timeline dropdown → **＋** → type `Q3 Roadmap`, Enter. Click the **🔗** share icon → enter a teammate's email → pick **Editor/Viewer** → confirm, close.

**VO:**
> "DMS Timeline Gantt — a real-time, shareable roadmap board. I'll create a timeline, 'Q3 Roadmap', and share it with a teammate as an editor or a read-only viewer. It syncs live."

---

### Scene 2 — Swim lanes & subgroups (0:18–0:40)
**On screen:** **＋ Add ▾ → Swim Lane** → `Taichi` (pick a color). Then **＋ Add ▾ → Subgroup** → `HB2 — HeartBeat2` under Taichi.

**VO:**
> "A swim lane is a project — here, Taichi. A subgroup is a workstream inside it: we'll add the HeartBeat2 service pack, 'HB2'. So swim lane equals the project, subgroup equals a service pack within it."

---

### Scene 3 — Add items & place them (0:40–1:05)
**On screen:** **＋ Add ▾ → Task** `API hardening` → click calendar for start, click again for end. Then **right-click** a date in the lane → **Add Milestone here** → `HB2 GA`, Enter. Drag the bar to reschedule.

**VO:**
> "Two ways to add work. From the Add menu — create a task, then click to set its start and end on the calendar. Or right-click any date to drop a task or milestone exactly there. Drag to reschedule."

---

### Scene 4 — Views (1:05–1:22)
**On screen:** Click **☀ Day → 📅 Week → 🗓 Month → 📆 Year**, pausing briefly on each.

**VO:**
> "Same plan, any zoom — Day and Week for detail, Month for the quarter, Year for the long-range roadmap."

---

### Scene 5 — Outro (1:22–1:30)
**On screen:** Hover a bar (tooltip) → click **⬇ PDF**.

**VO:**
> "Hover for details, and export a clean PDF to share. Plan it, share it, ship it."

---

### UI quick-reference
| Action | Where |
|---|---|
| New timeline / Share | dropdown **＋** / **🔗** icon |
| Add anything | **＋ Add ▾** menu |
| Quick add | **right-click** a date cell |
| Place task / milestone | click-start + click-end / single click |
| Views | **☀ Day · 📅 Week · 🗓 Month · 📆 Year** |
| Export / Undo | **⬇ PDF** (bottom-left) / **Ctrl+Z** |
