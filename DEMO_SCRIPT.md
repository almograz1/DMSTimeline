# DMS Timeline Gantt — 90-Second Demo (Google Vids shot list)

**Target:** ≤ 1:30 · **One table row = one Vids scene.**

### How to use this in Google Vids
1. **Record the app first** (Win + Alt + R, or Loom) on `https://dms-timeline.vercel.app`, doing the actions in the **Visual** column. Keep it loose — you'll trim per scene.
2. In Google Vids → new project → add **one scene per row** below.
3. Per scene: drop in the matching clip/screenshot · paste the **TTS line** into the scene's **voiceover / text-to-speech** field · type the **Caption** as on-screen text.
4. Trim each scene to its time, add auto-captions, export 1080p.

**Prep names:** swim lane `Taichi` · subgroup `HB2 — HeartBeat2` · task `API hardening` · milestone `HB2 GA`.
**Voice suggestion:** a calm, confident narrator; ~150 wpm. Total ≈ 200 words.
